# 💰 Expense Tracker Pro

A full-featured, offline-first personal expense tracker with **multi-currency support**, **monthly budgeting**, **remaining balance tracking**, and **top spending sector identification**. All data is stored locally in your browser—no sign-up, no servers, complete privacy.

## 🚀 Live Demo
[Click here to try it live!](https://your-username.github.io/expense-tracker)

## ✨ Features
- **Multi-Currency Support** – Choose from USD, EUR, GBP, INR, JPY, AUD, CAD, BRL, ZAR. All amounts instantly reformat.
- **Monthly Budget** – Set your total monthly income or budget.
- **Remaining Balance** – See exactly how much you have left. Turns **red** if you're over budget!
- **Top Spending Sector** – Automatically identifies which category you spent the most on (with emoji and amount).
- **Add/Delete Expenses** – Full CRUD operations with category selection.
- **Pie Chart Visualization** – Interactive doughnut chart showing spending breakdown by category.
- **Persistent Storage** – All data is saved to LocalStorage; survives browser restarts.
- **Sample Data** – Pre-loaded with sample expenses to showcase the UI instantly.
- **Dark Glass-morphism UI** – Sleek, modern, and easy on the eyes.

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox, Grid, glass-morphism)
- Vanilla JavaScript (ES6+)
- Chart.js (doughnut chart)
- LocalStorage API
- Intl.NumberFormat (for currency formatting)
