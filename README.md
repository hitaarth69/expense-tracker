# 💰 Expense Tracker

A clean, offline-first personal expense tracker that stores all your data locally in your browser. Track your spending, visualize categories with a pie chart, and stay on top of your finances—no sign-up required.

## 🚀 Live Demo
[Click here to try it live!](https://your-username.github.io/expense-tracker)

## ✨ Features
- **Add Expenses** – Description, amount, and category (Food, Transport, Entertainment, etc.).
- **Real-time Summary** – Total spending, highest spending category, and total entries.
- **Interactive Pie Chart** – Visual breakdown of spending by category using Chart.js.
- **Persistent Storage** – All data is saved to LocalStorage; your expenses remain even after closing the browser.
- **Delete & Clear All** – Remove individual expenses or wipe all data with a confirmation dialog.
- **Sample Data** – Pre-loaded with sample expenses so the page looks great the first time you open it.
- **Dark Theme** – Easy on the eyes, built with the same glass-morphism style as my other projects.

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox, Grid, glass-morphism)
- Vanilla JavaScript (ES6+)
- Chart.js (for the doughnut chart)
- LocalStorage API (for data persistence)
